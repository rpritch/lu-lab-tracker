## Installation

You can install `lu-lab-tracker` via [pip]:

```bash
pip install lu-lab-tracker
```

If napari is not already installed, you can install `lu-lab-tracker` with napari and Qt via:

```bash
pip install "lu-lab-tracker[all]"
```


To install latest development version:

```bash
pip install git+https://github.com/rpritch/lu-lab-tracker.git
```
